# Erick Xavier — CV React App

Currículo interativo em React com toggle PT/EN, pronto para deploy no Vercel.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Deploy no Vercel

### Opção 1 — Via GitHub (recomendado)

1. Crie um repositório no GitHub e faça push deste projeto
2. Acesse [vercel.com](https://vercel.com) e clique em **Add New Project**
3. Conecte seu repositório
4. As configurações já são detectadas automaticamente (Vite + React)
5. Clique em **Deploy** ✅

### Opção 2 — Via Vercel CLI

```bash
npm install -g vercel
vercel
```

## Build

```bash
npm run build
```

O projeto será gerado na pasta `dist/`.
